# iriscli gov pull-params

## 描述

生成param.json文件

## 使用方式

```
iriscli gov pull-params [flags]
```
打印帮助信息:

```
iriscli gov pull-params --help
```
## 标志

| 名称, 速记       | 默认值                      | 描述                                                                                                                                                 | 是否必须  |
| --------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| --path          | $HOME/.iriscli                | [string] iriscli home目录                                                                                                                      

## 示例

### 生成参数文件

```shell
iriscli gov pull-params
```

执行该指令，你会收到如下提示信息：

```txt
Save the parameter config file in  /Users/trevorfu/.iriscli/params.json
```
