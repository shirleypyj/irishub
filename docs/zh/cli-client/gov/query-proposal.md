# iriscli gov query-proposal

## 描述

查询单个提议的详情

## 使用方式

```
iriscli gov query-proposal [flags]
```
打印帮助信息:

```
iriscli gov query-proposal --help
```

## 标志

| 名称, 速记       | 默认值                      | 描述                                                                                                                                                 | 是否必须  |
| --------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| --proposal-id   |                            | [string] 提议ID                                                                                                        | Yes      |
## 示例

### 查询指定的提议

```shell
iriscli gov query-proposal --chain-id=test --proposal-id=1
```
