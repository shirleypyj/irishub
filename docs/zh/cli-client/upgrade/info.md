# iriscli upgrade info

## 描述

查询软件版本信息和升级模块的信息

## 用法

```
iriscli upgrade info
```

打印帮助信息:

```
iriscli upgrade info --help
```
## 标志

| 名称, 速记       | 默认值                     | 描述                                                        | 必需     |
| --------------- | -------------------------- | ----------------------------------------------------------------- | -------- |
